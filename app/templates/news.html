First public available version

Another big day of this project. Today I will launch first public available v1.1, I hope my little server will handle incredible 5 users lol. Some changes happened since closed v1.0 was launched:

  • added ability to check password in registration, login and password restore windows
  • new cryptocurrency added: MANTRA DAO, Aptos, BNB, Bitget token, OKB
  • updated aliens with better quality of images
  • added caching in redis data for charts
  • added client side caching support for static content
  • added ability to change avatar
  • added vendor prefixes with postcss autoprefixer in css files
  • done some changes in ui design
  • fixed some bugs

Closed beta

Finally this day has come. Many hours of development, testing and debugging behind. In last month I have changed api for retrieving prices and other crypto data, finished new design that definitely better than old one. And now available these features: sign in/up, password restore, active session management, data of 20 most popular cryptocurrencies in real time, ability to trade cryptocurrency, cryptowallet data and statistics in profile, transaction history. From this moment CryptoDeal v1.0 available at cryptodeal.pw. Now it is closed beta, several guys have access in order to test site, but v1.1(planned in 1-2 weeks) will be public and everyone will be able to use it.
