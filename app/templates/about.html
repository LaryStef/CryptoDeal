CryptoDeal

CryptoDeal is web application that allows you to explore cryptocurrency(and securities in future) without any real investments. You can see the list of cryptocurrencies, their prices, and other information. Profile gives you possibility to track changes in your cryptowallet worth. The data is updated every hour. Also every new account has lifting 1M$ to start investing, don't fun, invest wisely. The application uses the CoinLore API to get the data. Technologies used: python, javascript, postgresql, redis, flask, sqlalchemy, docker, nginx, celery, poetry, webpack, css, scss, html, jinja.
